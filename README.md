# G O C I A

**G**lobal **O**ptimizer for **C**lusters, **I**nterfaces, and **A**dsorption

```GOCIA``` is a genetic algorithm toolkit and Python modules specialized for simulating restructured and adsorbate-covered surfaces.

## Requirements

- Python 3.6 or later
- NumPy
- SciPy
- ASE
